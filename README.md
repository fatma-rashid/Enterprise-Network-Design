# Enterprise Network Design

## Overview
A multi-branch enterprise network implemented using Cisco Packet Tracer.

## Features
- OSPF Dynamic Routing
- EtherChannel Link Aggregation
- Spanning Tree Protocol (STP)
- VLAN Segmentation
- Department-based Network Design
- Redundant Switching Architecture
- Multi-site Connectivity
- Server Integration

## Technologies
- Cisco Routers 2911
- Cisco Catalyst Switches
- OSPF
- STP
- EtherChannel
- VLANs

## Departments
- HR
- IT
- Call Center
- Engineering
- Sales
- Finance
- Accounting
